# LCD Look Generator

A browser-based template to create LCD-style text overlays.  
Built with plain HTML5 Canvas + JavaScript.

## Features
- Multiline text input (textarea)
- Adjustable font size, tracking, and line spacing
- Left / center / right alignment toggle
- Auto word wrap (set wrap width %)
- LCD effects: scanlines, RGB fringing, bloom/blur, vignette, noise grain
- Export PNG with one click
